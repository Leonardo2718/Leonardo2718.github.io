Projects

In my spare time, I often work on various personal projects. Here is a list of my bigger programs.

Lepton

Lepton is a small text editor written in C++ with Qt. It is based on the QScintilla editor library and uses a custom lexer for syntax highlighting. It supports syntax highlighting for various programming and markup languages including C++, XML, Python, and Haskell. It also supports some very basic project management and is highly customizable.

Quark

Quark is a Markdown based note taking program. It is written in Python and uses PyQt. It supports many features including: basic note management, syntax highlighting in the Markdown editor (using a custom lexer), spell checking, a live preview, synchronized scrolling of the editor and the preview, MathJax support, code blocks, syntax highlighting for code blocks, custom CSS, custom HTML output templates, and exporting notes to HTML files.

OGLA

OGLA is a generic C++ lexical analysis (lexer) library. It's intended for use in applications which require basic lexical analysis for simple tasks such as configuration file parsing and pretty-printing. It is based on regular expressions and allows grammars to be loaded dynamically. It is not intended to be a very fast lexer, but rather to be as generic and flexible as possible. The main implementation is completely stand-alone as it only uses the standard C++ libraries. However, there is also an alternative (and somewhat experimental) implementation that uses Boost Regex, which is more powerful than the standard regular expression library.

tuc

tuc is a small, experimental, and primitive compiler. It is purely intended as an educational project. It has no real practical application other than being a fun environment for testing ideas and experimenting. It compiles a language I created called U (TUC obviously stands for The U Compiler).
